# 🤖 AI智能客服系统

## 📌 项目简介

本项目是一个基于 Python 实现的轻量级 AI 客服系统示例，能够根据用户输入自动生成回复，用于模拟电商场景中的常见客服对话（如订单查询、退款咨询等）。

系统采用规则驱动的方式实现基础对话逻辑，具备简单的交互能力，适合作为 AI 客服或智能对话系统的入门项目。

---

## 🚀 功能特点

* 支持基础中文对话（问候 / 订单 / 退款 / 发货等）
* 基于规则的智能回复机制
* 命令行实时交互
* 结构简单，易于扩展为更复杂的AI系统

## 💬 使用示例

```text
=== AI客服系统已启动（输入 exit 退出）===
你：你好
AI：您好！我是AI客服，请问有什么可以帮您？

你：查询订单
AI：请提供订单号，我帮您查询订单状态。

你：我要退款
AI：您可以在订单页面申请退款，我可以帮您查看流程。
```

输入 `exit` 可退出系统
